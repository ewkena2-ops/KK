# Klever Kuche Website

Static one-page website for Klever Kuche in Addis Ababa, Ethiopia.

## Netlify Deploy

- Connect this repository from GitHub.
- Production branch: `main`
- Build command: leave empty
- Publish directory: `.`

After the first successful deploy, add or buy the domain in Netlify under **Domain management**.

## Cloudflare Pages Deploy

- Connect this repository from GitHub in **Workers & Pages**.
- Project type: **Pages**
- Production branch: `main`
- Framework preset: **None**
- Build command: leave empty
- Build output directory: `.`

After deploy, add `kleverkuche.com` as a custom domain in Cloudflare Pages and move DNS from Netlify to Cloudflare.
